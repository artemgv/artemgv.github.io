# ℹ️ О Mastering Words — карточки для запоминания английских слов и фраз


**Mastering Words** — это сайт, созданный для изучения английских слов и выражений по флэш-картам. В его основе лежат колоды, ставшие популярными на **Quizlet**.

Уникальной особенностью сайта является интеграция каждой колоды с орфографическим тренажером: встроенный в страницу колоды виджет **Quizlet** позволяет тренировать правописание слов быстро, удобно и без лишних кликов.

# Почему Quizlet — это только начало

Многие начинают с **Quizlet** — и не зря. Он прост, работает прямо в браузере и не требует регистрации. Кажется, что это идеальный способ учить слова.

Но где подвох? Если всё так удобно, почему сервис бесплатный и работает даже без учётной записи?

Всё просто: **в Quizlet нет системы интервальных повторений (SRS — Spaced Repetition System)**, которая отслеживает ваш прогресс и в нужный момент предлагает нужные карточки для повторения.
Отсутствие активной SRS-системы, требующей вычислительных ресурсов на стороне сервера, делает обслуживание незарегистрированных пользователей для **Quizlet** практически бесплатным.

# Что делает SRS

Система интервальных повторений для каждого пользователя и каждой его карточки **автоматически подбирает оптимальный момент для повтора.**
> Карточка показывается именно тогда, когда пользователь **почти готов её забыть** — такая мнемотехника в разы усиливает закрепление в памяти.
{: .prompt-tip }

Пользователи **Anki**, где есть SRS, запоминают в 5–7 раз больше слов при той же интенсивности занятий.
**Anki** не просто помогает выучить слова — **Anki помогает их не забыть**.

# Почему классический Anki неудобен

**Anki** открывает гораздо больше возможностей, но на старте всё кажется запутанным и сложным.
Попробуйте загрузить в классический **Anki** суперколоду — и убедитесь, как легко потеряться среди тысяч карточек, ручных фильтров и настроек.
В общем, **Anki из коробки — это путь самурая.**

Чтобы упростить процесс, мы разбили суперколоду на множество групп по 33–37 слов.

# Как работает система

Представьте, что каждый день вы учите новую группу в режиме зубрёжки — как в **Quizlet**, но в Android-приложении. Например, сегодня вы проходите группу №56.

В дополнение к её словам сервер **Anki** сегодня «подмешает»:
- слова из ранее изученных групп №1–55 (в режиме SRS),
- и все сложные слова, которые вы отметили флажками в этих группах.

> Таким образом, вы каждый день просто проходите дневную норму, а ваш сервер **Anki** готовит микс из свежих, старых и сложных слов — без ручных фильтров и лишней рутины.
{: .prompt-tip }

# Режим чистой зубрёжки

При желании вы можете временно перейти в режим чистой зубрёжки, отправив серверу синхронизации команду о переходе к нужной группе, чтобы повторить именно этот набор карточек.

Кроме того, через параметры SRS-колоды можно подкорректировать дневную норму.

Эти возможности описаны в разделе «Помощь» — они не нужны всем, но полезны продвинутым пользователям.

# Push-уведомления через AnkiDroid Notifications

Дополнительно можно установить приложение AnkiDroid Notifications. После каждого перестроения фильтрованных колод сервер синхронизации будет отправлять в него push-уведомление с текстом вроде:
«Вы перешли к изучению группы №57».

Такое уведомление будет отображаться в шторке Android, напоминая, какую группу карточек вы сейчас проходите и что пора заняться учебой.

При тапе на уведомлении откроется AnkiDroid, и вам останется только нажать кнопку «Синхронизировать», чтобы увидеть обновлённые колоды.

# Личный сервер делает всё автоматически

Чтобы **Anki** работал без ручных настроек, вам понадобится свой личный сервер синхронизации, работающий 24/7 на крошечном VPS.
Он будет полностью принадлежать вам, хранить только ваш прогресс и автоматически создавать резервные копии.

Если вы решите сделать перерыв и временно прекратить аренду VPS, достаточно забрать резервную копию с сервера — позже можно будет восстановить весь прогресс на новом сервере.

# Развёртывание и запуск

Для развёртывания личного сервера синхронизации **Anki** достаточно арендовать VPS с **Windows Server Core 2019 (ENG)** в конфигурации **UltraLite** (1 CPU, 512 MB RAM, 10 GB HDD) и запустить на нём одну единственную команду.

Установка занимает около часа и проходит полностью автоматически.
